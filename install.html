如何安裝 PyPtt¶

Python 版本¶

建議你使用 CPython 3.8 以上的版本。
因為我們僅支援 CPython 3.8 或更新的版本。

相依套件¶

PyPtt 目前相依於以下套件，這些套件都會在安裝的過程中被自動安裝。

  • websockets is a library for building WebSocket servers and clients in Python with a focus on correctness, simplicity, robustness, and performance.

  • uao is a pure Python implementation of the Unicode encoder/decoder.

  • SingleLog is a simple logging library for Python.

  • requests is a Python HTTP library, released under the Apache License 2.0.

  • AutoStrEnum is a Python library that provides an Enum class that automatically converts enum values to and from strings.

使用虛擬環境安裝 (建議)¶

我們建議各位使用虛擬環境來安裝 PyPtt，因為如此可以最大程度地避免套件衝突。

你可以從 Virtual Environments and Packages 中了解，更多關於使用虛擬環境的理由與如何建立你的虛擬環境。

從 PyPi 安裝¶

你可以使用以下指令來安裝 PyPtt

pip install PyPtt

現在 PyPtt 已經成功安裝了，來看看 PyPtt 的 API 說明 或者 使用範例 吧！
