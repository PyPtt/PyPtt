參數型態#

這裡介紹 PyPtt 的參數型態

HOST#

  • 連線的 PTT 伺服器。

PyPtt.HOST.PTT1#

批踢踢實業坊

PyPtt.HOST.PTT2#

批踢踢兔

Language#

  • 顯示訊息的語言。

PyPtt.Language.MANDARIN#

繁體中文

PyPtt.Language.ENGLISH#

英文

ConnectMode#

  • 連線的模式。

PyPtt.ConnectMode.WEBSOCKETS#

使用 WEBSOCKETS 連線

PyPtt.ConnectMode.TELNET#

使用 TELNET 連線

NewIndex#

  • 搜尋 Index 的種類。

PyPtt.NewIndex.BOARD#

搜尋看板 Index

PyPtt.NewIndex.MAIL#

搜尋信箱 Index

SearchType#

  • 搜尋看板的方式。

PyPtt.SearchType.KEYWORD#

搜尋關鍵字

PyPtt.SearchType.AUTHOR#

搜尋作者

PyPtt.SearchType.COMMENT#

搜尋推文數

PyPtt.SearchType.MARK#

搜尋標記

PyPtt.SearchType.MONEY#

搜尋稿酬

ReplyTo#

  • 回文的方式。

PyPtt.ReplyTo.BOARD#

回文至看板

PyPtt.ReplyTo.MAIL#

回文至信箱

PyPtt.ReplyTo.BOARD_MAIL#

回文至看板與信箱

CommentType#

  • 推文方式。

PyPtt.CommentType.PUSH#

推

PyPtt.CommentType.BOO#

噓

PyPtt.CommentType.ARROW#

箭頭

PostStatus#

  • 文章狀態。

PyPtt.PostStatus.EXISTS#

文章存在

PyPtt.PostStatus.DELETED_BY_AUTHOR#

被作者刪除

PyPtt.PostStatus.DELETED_BY_MODERATOR#

被板主刪除

PyPtt.PostStatus.DELETED_BY_UNKNOWN#

無法判斷，被如何刪除

MarkType#

  • 版主標記文章種類

PyPtt.MarkType.S#

S 文章

PyPtt.MarkType.D#

標記文章

PyPtt.MarkType.DELETE_D#

刪除標記文章

PyPtt.MarkType.M#

M 起來

PyPtt.MarkType.UNCONFIRMED#

待證實文章

UserField#

  • 使用者資料欄位。

PyPtt.UserField.ptt_id#

使用者 ID

PyPtt.UserField.money#

經濟狀態

PyPtt.UserField.login_count#

登入次數

PyPtt.UserField.account_verified#

是否通過認證

PyPtt.UserField.legal_post#

文章數量

PyPtt.UserField.illegal_post#

退文數量

PyPtt.UserField.activity#

目前動態

PyPtt.UserField.mail#

信箱狀態

PyPtt.UserField.last_login_date#

最後登入時間

PyPtt.UserField.last_login_ip#

最後登入 IP

PyPtt.UserField.five_chess#

五子棋戰積

PyPtt.UserField.chess#

象棋戰積

PyPtt.UserField.signature_file#

簽名檔

CommentField#

  • 推文資料欄位。

PyPtt.CommentField.type#

推文類型，推噓箭頭，詳見 CommentType

PyPtt.CommentField.author#

推文作者

PyPtt.CommentField.content#

推文內容

PyPtt.CommentField.ip#

推文 IP (如果存在)

PyPtt.CommentField.time#

推文時間

FavouriteBoardField#

  • 我的最愛資料欄位。

PyPtt.FavouriteBoardField.board#

看板名稱

PyPtt.FavouriteBoardField.title#

看板標題

PyPtt.FavouriteBoardField.type#

類別

MailField#

  • 信件資料欄位。

PyPtt.MailField.origin_mail#

原始信件全文

PyPtt.MailField.author#

信件作者

PyPtt.MailField.title#

信件標題

PyPtt.MailField.date#

信件日期

PyPtt.MailField.content#

信件內容

PyPtt.MailField.ip#

信件 IP

PyPtt.MailField.location#

信件位置

PyPtt.MailField.is_red_envelope#

是否為紅包

BoardField#

  • 看板資料欄位。

PyPtt.BoardField.board#

看板名稱

PyPtt.BoardField.online_user#

在線人數

PyPtt.BoardField.chinese_des#

看板中文名稱

PyPtt.BoardField.moderators#

看板板主清單

PyPtt.BoardField.open_status#

看板公開狀態，是否隱板

PyPtt.BoardField.into_top_ten_when_hide#

隱板時是否可以進入十大排行榜

PyPtt.BoardField.can_non_board_members_post#

非看板成員是否可以發文

PyPtt.BoardField.can_reply_post#

是否可以回覆文章

PyPtt.BoardField.self_del_post#

是否可以自刪文章

PyPtt.BoardField.can_comment_post#

是否可以推文

PyPtt.BoardField.can_boo_post#

是否可以噓文

PyPtt.BoardField.can_fast_push#

是否可以快速推文

PyPtt.BoardField.min_interval_between_comments#

推文間隔時間

PyPtt.BoardField.is_comment_record_ip#

是否記錄推文 IP

PyPtt.BoardField.is_comment_aligned#

推文是否對齊

PyPtt.BoardField.can_moderators_del_illegal_content#

板主是否可以刪除違規文字

PyPtt.BoardField.does_tran_post_auto_recorded_and_require_post_permissions#

是否自動記錄轉錄文章並需要發文權限

PyPtt.BoardField.is_cool_mode#

是否為冷板模式

PyPtt.BoardField.is_require18#

是否為 18 禁看板

PyPtt.BoardField.require_login_time#

發文需要登入次數

PyPtt.BoardField.require_illegal_post#

發文需要最低退文數量

PyPtt.BoardField.post_kind_list#

發文類別，例如 [公告] [問卦] 等

PostField#

  • 文章資料欄位。

PyPtt.PostField.board#

文章所在看板

PyPtt.PostField.aid#

文章 ID，例如：#1Z69g2ts

PyPtt.PostField.index#

文章編號，例如：906

PyPtt.PostField.author#

文章作者

PyPtt.PostField.date#

文章日期

PyPtt.PostField.title#

文章標題

PyPtt.PostField.content#

文章內容

PyPtt.PostField.money#

文章稿酬，P 幣

PyPtt.PostField.url#

文章網址

PyPtt.PostField.ip#

文章 IP

PyPtt.PostField.comments#

文章推文清單，詳見 CommentField

PyPtt.PostField.post_status#

文章狀態，詳見 PostStatus

PyPtt.PostField.list_date#

文章列表日期

PyPtt.PostField.has_control_code#

文章是否有控制碼

PyPtt.PostField.pass_format_check#

文章是否通過格式檢查

PyPtt.PostField.location#

文章 IP 位置

PyPtt.PostField.push_number#

文章推文數量

PyPtt.PostField.is_lock#

文章是否鎖定

PyPtt.PostField.full_content#

文章完整內容

PyPtt.PostField.is_unconfirmed#

文章是否為未確認文章
