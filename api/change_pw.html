change_pw¶

PyPtt.API.change_pw(self, new_password: str) → None

更改密碼。

參數:

new_password (str) – 新密碼。

回傳:

None

引發:

  • PyPtt.RequireLogin – 需要登入。

  • PyPtt.SetConnectMailFirst – 需要先設定聯絡信箱。

  • PyPtt.WrongPassword – 密碼錯誤。

範例:

import PyPtt

ptt_bot = PyPtt.API()
try:
    # .. login ..
    ptt_bot.change_pw(new_password='123456')
    # .. do something ..
finally:
    ptt_bot.logout()
