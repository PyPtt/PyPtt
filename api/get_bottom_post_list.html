get_bottom_post_list¶

PyPtt.API.get_bottom_post_list(self, board: str) → List[str]

取得看板置底文章清單。

參數:

board (str) – 看板名稱。

回傳:

List[post]，置底文章清單，詳見 PostField。

引發:

  • PyPtt.RequireLogin – 需要登入。

  • PyPtt.NoSuchBoard – 看板不存在。

範例:

import PyPtt

ptt_bot = PyPtt.API()

try:
    # .. login ..
    bottom_post_list = ptt_bot.get_bottom_post_list(board='Python')
    # .. do something ..
finally:
    ptt_bot.logout()
