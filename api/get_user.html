get_user#

PyPtt.API.get_user(self, user_id: str) → Dict

取得使用者資訊。

參數:

user_id (str) – 使用者 ID。

回傳:

Dict，使用者資訊。詳見 使用者資料欄位

引發:

  • RequireLogin – 需要登入。

  • NoSuchUser – 使用者不存在。

範例:

import PyPtt

ptt_bot = PyPtt.API()
try:
    # .. login ..
    user_info = ptt_bot.get_user('CodingMan')
    # .. do something ..
finally:
    ptt_bot.logout()
