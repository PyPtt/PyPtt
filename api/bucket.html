bucket¶

PyPtt.API.bucket(self, board: str, bucket_days: int, reason: str, ptt_id: str) → None

水桶。

參數:

  • board (str) – 看板名稱。

  • bucket_days (int) – 水桶天數。

  • reason (str) – 水桶原因。

  • ptt_id (str) – PTT ID。

回傳:

None

引發:

  • RequireLogin – 需要登入。

  • UnregisteredUser – 未註冊使用者。

  • NoSuchBoard – 看板不存在。

  • NoSuchUser – 使用者不存在。

  • NeedModeratorPermission – 需要看板管理員權限。

範例:

import PyPtt

ptt_bot = PyPtt.API()
try:
    # .. login ..
    ptt_bot.bucket(board='Test', bucket_days=7, reason='PyPtt 程式水桶測試', ptt_id='test')
    # .. do something ..
finally:
    ptt_bot.logout()
