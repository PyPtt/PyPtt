未來開發計劃¶

  • WebSocket 支援 Tor or Proxy

    期待有一天可以透過 Tor 或 Proxy 來連接到 PTT，讓 PyPtt 可以自由地在雲端伺服器運作。

  • PyPtt Service image
    期待 PyPtt 在未來可以有 API 形式的服務，讓大家可以透過 API 呼叫來使用 PyPtt。
    這樣其實在某個層面也上可以達到使用 Tor or Proxy 的目的。
